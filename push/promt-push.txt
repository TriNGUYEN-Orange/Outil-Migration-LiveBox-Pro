### ÉTAPE 1 : Wifi
1. Cliquer #sah_footer > div > div.iconbar-button.icon.icon-network.entered
2. Cliquer #wifiAdvanced
Dans le conteneur #wifi_accesspoint_form, il faut récupérer 3 éléments dont un de chaque est accèssible par son lien, quand on ouvert ce lien, on a des infos sur le wifi.
Pour les 2 premiers liens, ce sont des wifi 2.4 et 5 donc ils ont le même structure de page, même instruction pour push des données.
Dans le 3e lien, c'est le wifi invité donc pas la même structure.
3. Pour les 2e premiers liens, voici les étapes à faire
    3.1 regarder l'état activé de button #wifi_accesspoint24 ou #wifi_accesspointboth (dans le cas pas differencier le 2.4 et le 5), si c'est "ON", ne fait rien, si c'est "OFF", cliquer sur le button pour allumer
    3.2 cliquer #wifi_accesspoint24_link ou #wifi_accesspointboth pour aller dans la page de configuration de wifi 2.4
    3.2 regarder dans json "wifi" -> "ssid", puis push l'info dans #wifi_private_ssid
    3.3 regarder dans json "wifi" -> "mot_de_passe", puis #wifi_private_securitykey
    3.4 regarder dans json "wifi" -> "wifi2_4" -> "diffusion_ssid" si c'est true/false, puis, vérifier #wifi_private_broadcastssid_id_1 checkbox s'il est checked ou pas pour mettre le même choix
    3.5 regarder dans json "wifi" -> "differencier_reseaux" true/false, puis faire #wifi_private_different_ssid (select) toujours mettre 0 oui
    3.6 regarder dans json "wifi" -> "wifi2_4" -> "afficher_cle_ecran" true/false, puis #wifi_private_oledSecurityKey_1 (checkbox)
    3.7 cliquer sur lien #advanced_parameters_link
    3.8 regarder dans json "wifi" -> "wifi2_4" -> "mode", puis #wifi_private_mode24 (select) b/g/n/bg/gn/bgn/ax , ne pas laisser minuscule ou majuscule concernent le résultat
    3.9 regarder dans json "wifi" -> "wifi2_4" -> "filtrage_mac" false désactivé / true activé, puis #wifi_private_macfiltering_enable (select) 1 activé / 0 désactivé
    3.10 cliquer #save 
    3.11 cliquer #popup_confirm_submit
    3.12 cliquer #wifi_accesspoint5_link_txt
    3.13 regarder dans json "wifi" -> "wifi5" -> "ssid", puis #wifi_private_ssid (text)
    3.14 regarder dans json "wifi" -> "wifi5" -> "diffusion_ssid", puis #wifi_private_broadcastssid_id_1 (checkbox)
    3.15 regarder dans json "wifi" -> "wifi5" -> "mdp", puis #wifi_private_securitykey (text)
    3.16 mettre toujours Oui pour #wifi_private_oledSecurityKey_1 (checkbox)
    3.17 regarder dans json "wifi" -> "differencier_reseaux" true/false, puis faire #wifi_private_different_ssid (select) 0 oui, 1 non
    3.18 regarder dans json "wifi" -> "wifi5" -> "filtrage_mac" false désactivé / true activé, puis #wifi_private_macfiltering_enable (select) 1 activé / 0 désactivé
    3.19 cliquer #save
    3.20 cliquer #popup_confirm_submit
